# PocketBase Access

You can access your PocketBase admin panel by visiting the page after deployment.

```
https://yourpocketbasedomain.com/_/
```

## Notable Variables

| Variable Name | Service Name | Description |
|---------------|--------------|-------------|
| `PB_ADMIN_EMAIL` | PocketBase | The email to login to the PocketBase admin panel |
| `PB_ADMIN_PASSWORD` | PocketBase | The password to login to the PocketBase admin panel |

## Accessing your Supabase Admin UI

1. Visit the URL of the `PocketBase` service - be sure to add `/_/` to the end of the URL
2. Login with the credentials from the `Variables` section of the `PocketBase` service
