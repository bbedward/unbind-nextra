# MeiliSearch Access

You can access your MeiliSearch dashboard by visiting the page after deployment.

## Notable Variables

| Variable Name | Service Name | Description |
|---------------|--------------|-------------|
| `MEILI_MASTER_KEY` | MeiliSearch | The master key to access Meilisearch |

## Using Meilisearch

Reference the [Meilisearch documentation](https://www.meilisearch.com/docs/learn/resources/sdks) for more information on how to use Meilisearch, SDKs should be configured with the `MEILI_MASTER_KEY` variable value in the `MeiliSearch` service.
