## Notable Variables

| Variable Name | Service Name | Description |
|---------------|--------------|-------------|
| `MINIO_ROOT_USER` | MinIO | The username to login to the MinIO console |
| `MINIO_ROOT_PASSWORD` | MinIO | The password to login to the MinIO console |

## Accessing your MinIO Console

The MinIO template also deploys the MinIO web interface, in addition to the MinIO API service.

1. Visit the URL of the `MinIO` service
2. Login with the credentials from the `Variables` section of the `MinIO` service
