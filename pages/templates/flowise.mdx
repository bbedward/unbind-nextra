# Flowise Access

You can access your Flowise dashboard by visiting the page after deployment.

## Notable Variables

| Variable Name | Service Name | Description |
|---------------|--------------|-------------|
| `FLOWISE_USERNAME` | Flowise | The username to login to the Flowise dashboard |
| `FLOWISE_PASSWORD` | Flowise | The password to login to the Flowise dashboard |

## Accessing your Flowise Dashboard

1. Visit the URL of the `Flowise` service
2. Login with the credentials from the `Variables` section of the `Flowise` service
