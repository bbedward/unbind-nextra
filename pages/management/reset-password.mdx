# Reset Password

<Callout type="warning">
Unbind's user management is in active development, this is a "disaster recovery" guide for resetting your admin password.
</Callout>

## Gain access to the pod
This will get you a terminal into the Unbind API pod/container.

```bash copy
kubectl exec -it -n unbind-system $(kubectl get pods -n unbind-system -l app=unbind-api -o jsonpath='{.items[0].metadata.name}') -- /bin/bash
```

## Reset the password

```bash copy
/app/cli user:change-password --email=<YOUR_EMAIL> --password=<NEW_PASSWORD>
```

## Login
Log in to your Unbind dashboard and verify the change went into affect.
